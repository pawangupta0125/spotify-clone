# Spotify Clone

A Spotify-inspired front-end clone created using HTML and CSS.

## Features
- Responsive layout
- Sidebar navigation
- Music cards and playlists section
- Sticky navigation bar
- Clean and modern UI inspired by Spotify

## Technologies Used
- HTML5
- CSS3

## Future Improvements
- Add JavaScript functionality
- Music playback controls
- Dynamic playlist generation
- Search feature
- Dark/Light mode toggle

## Purpose
This project was built to improve my front-end development skills and gain hands-on experience with HTML and CSS layouts.
